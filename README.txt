getEmailaddr是一个用PHP写的在线邮件地址收集器，最初是给豆瓣网邀请码分享小组的邀请码分享者写的一个小工具。
使用Curl获取网页html文档，并使用使用正则表达获得文档中的邮箱地址，原理很简单，没什么技术含量。大家不要见笑，只是想学学Git。

有喜欢看源码的，就拉下去看吧。想完善更是欢迎。
